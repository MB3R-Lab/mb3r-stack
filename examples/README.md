# Examples

These examples are intentionally small and operator-editable.

- `local-demo/` enables a local evaluation footprint.
- `ci-demo/` shows an ephemeral CI-style installation.
- `minimal-production-eval/` shows conservative pinning and external registry overrides.
- `otel-demo/` contains values for integrating the OpenTelemetry Demo with the addon.
- `github/`, `gitlab/`, and `jenkins/` contain consumer CI examples.
